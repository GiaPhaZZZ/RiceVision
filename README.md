# RiceVision : Precision Diagnostics for Vietnam’s Rice Fields

RiceVision is a specialized Vision Language Model (VLM) designed to safeguard Vietnam’s agricultural heritage. By bridging the gap between advanced deep learning and local agronomy, the project provides a two-fold solution for farmers and agronomists:

Intelligent Diagnosis: Using real-time image analysis, the model identifies specific rice diseases—such as Blast, Sheath Blight, or Bacterial Leaf Streak—with high precision, even in varied lighting and field conditions.

Localized Advisory: Beyond detection, it acts as a digital consultant, offering pesticide recommendations strictly filtered by the Vietnam Ministry of Agriculture and Rural Development (MARD) regulations.

By integrating Vietnamese-specific chemical registries and safety protocols, RiceVision ensures that treatment is not only effective but also legally compliant and environmentally sustainable, helping secure a "Golden Harvest" for local communities.
